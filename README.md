# Sontha - Modern Messenger

Современный мессенджер с темной темой, регистрацией через email-код и поддержкой аудио/видео звонков через WebRTC.

## ✨ Особенности

- 🔐 **Регистрация через email** - верификация через 6-значный код
- 💬 **Мгновенные сообщения** - обмен сообщениями в реальном времени через Socket.IO
- 📞 **Аудио/видео звонки** - WebRTC для голосовых и видеозвонков
- 🎨 **Современный дизайн** - темная тема с плавными анимациями и градиентами
- 🖼️ **SVG иконки** - встроенные векторные иконки без внешних файлов
- 🚀 **Electron приложение** - нативное настольное приложение

## 📋 Требования

- Node.js (v18+)
- npm или yarn

## 🚀 Установка

1. Установите зависимости:
```bash
npm install
```

## 🎮 Запуск

### 1. Запустите сервер Socket.IO:
```bash
npm run server
```

Сервер запустится на `http://localhost:3000`

### 2. В новом терминале запустите Electron приложение:
```bash
npm start
```

## 📝 Использование

### Регистрация

1. Введите email, имя и username
2. Нажмите "Continue"
3. Введите 6-значный код из консоли сервера (в консоли сервера будет выведен код)
4. Нажмите "Verify"

> **Примечание**: Отправка email по умолчанию отключена для демо. Код верификации выводится в консоль сервера.

### Чаты

- После входа вы увидите список зарегистрированных пользователей
- Нажмите на пользователя, чтобы открыть чат
- Введите сообщение и нажмите Enter или кнопку отправки

### Звонки

- Откройте чат с пользователем
- Нажмите на иконку телефона для аудиозвонка
- Нажмите на иконку видеокамеры для видеозвонка
- Используйте кнопки управления для mute/unmute и включения/выключения камеры

## 🔧 Конфигурация email (для продакшена)

Для включения отправки email откройте `server/server.js` и настройте:

```javascript
const transporter = nodemailer.createTransport({
  host: 'smtp.gmail.com',
  port: 587,
  secure: false,
  auth: {
    user: 'your-email@gmail.com',      // Ваш email
    pass: 'your-app-password'          // App password из Google
  }
});
```

Раскомментируйте блок отправки email в обработчике `register:request`.

## 🎨 Технологии

- **Electron** - фреймворк для десктопных приложений
- **Socket.IO** - WebSocket для реального времени
- **WebRTC** (SimplePeer) - аудио/видео звонки
- **Node.js** - серверная часть
- **Nodemailer** - отправка email
